### 纯CSS3动画效果实现类技能CD加载效果
#### html
```html
<div class="iib-mark">
  <div class="l1 vam"></div>
  <div class="r1 vam"></div>
  <div class="l2"></div>
</div>
```
#### css
```css
.iib-mark {
  position: relative;
  top: 0;
  left: 0;
  width: 100rpx;
  height: 100rpx;
}
.iib-mark>div {
  width: 50%;
  height: 100%;
  background: #fff;
}
.iib-mark>.l1 {
  position: absolute;
  top: 0;
  left: 0;
  z-index: 3;
  transform-origin: right center;
}
.iib-mark>.l1.vam {
  background: #c8c8c8;
  animation: v1 1.5s linear;
  animation-fill-mode: forwards;
}
.iib-mark>.r1 {
  position: absolute;
  top: 0;
  right: 0;
  z-index: 1;
  transform-origin: left center;
}
.iib-mark>.r1.vam {
  background: #c8c8c8;
  animation: v2 1.5s linear 1.5s;
  animation-fill-mode: forwards;
}
.iib-mark>.l2 {
  position: absolute;
  top: 0;
  left: 0;
  z-index: 2;
  background: white;
}
@keyframes v1 {
  0% {
    transform: rotate(0);
  }
  99.9% {
    z-index: 3;
  }
  100% {
    transform: rotate(-180deg);
    z-index: -1;
  }
}
@keyframes v2 {
  0% {
    transform: rotate(0);
  }
  99.9% {
    opacity: 1;
  }
  100% {
    transform: rotate(-180deg);
    opacity: 0;
  }
}
```
