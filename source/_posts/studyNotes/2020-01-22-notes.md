#### nginx.conf
```conf
# ...
location /api {
  rewrite /api/(.+)$ /$1 break;
  proxy_pass http://localhost:3000 ;
  proxy_redirect off;
}
# ...
```
访问：`http://localhost/api`，会代理到`http://localhost:3000`，`http://localhost:3000`路由中无需额外写/api，nginx会自动重写。
